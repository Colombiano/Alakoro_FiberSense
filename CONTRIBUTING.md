# 🤝 Contribuindo / Contributing

Obrigado pelo interesse em contribuir com o Alakoro FiberSense!
Thank you for your interest in contributing to Alakoro FiberSense!

---

## 🚀 Como Contribuir / How to Contribute

### 1. Fork e Clone / Fork and Clone
```bash
git clone https://github.com/SEU_USUARIO/Alakoro_FiberSense.git
cd Alakoro_FiberSense
```

### 2. Crie um Branch / Create a Branch
```bash
git checkout -b feature/minha-feature
# ou / or
git checkout -b fix/minha-correcao
```

### 3. Instale em Modo Desenvolvimento / Install in Development Mode
```bash
pip install -r requirements.txt
pip install -e .
```

### 4. Execute os Testes / Run Tests
```bash
pytest tests/ -v
```

### 5. Commit e Push / Commit and Push
```bash
git add .
git commit -m "feat: descrição clara da mudança"
git push origin feature/minha-feature
```

### 6. Abra um Pull Request / Open a Pull Request

---

## 📝 Padrões de Código / Code Standards

- **PEP 8** — Estilo Python / Python style
- **Docstrings bilíngues** — PT + EN / Bilingual docstrings
- **Type hints** — Em todas as funções / On all functions
- **Testes** — Todo novo recurso precisa de testes / Every new feature needs tests

---

## 🏷️ Tipos de Commit / Commit Types

| Tipo / Type | Uso / Use |
|-------------|-----------|
| `feat:` | Novo recurso / New feature |
| `fix:` | Correção de bug / Bug fix |
| `docs:` | Documentação / Documentation |
| `test:` | Testes / Tests |
| `refactor:` | Refatoração / Refactoring |
| `perf:` | Performance / Performance |
| `chore:` | Manutenção / Maintenance |

---

## 📋 Checklist do PR / PR Checklist

- [ ] Testes passam / Tests pass
- [ ] Documentação atualizada / Documentation updated
- [ ] CHANGELOG.md atualizado / CHANGELOG.md updated
- [ ] Código segue PEP 8 / Code follows PEP 8
- [ ] Docstrings bilíngues / Bilingual docstrings
